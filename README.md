# Évaluation Git & GitHub
## Qu’est-ce que Git ?

Git est un système de gestion de versions qui permet de suivre les modifications d’un projet, de revenir en arrière si nécessaire et de travailler efficacement sur un même code.

## Qu’est-ce que GitHub ?

GitHub est une plateforme en ligne qui permet d’héberger des dépôts Git, de partager son code et de collaborer avec d’autres développeurs.
